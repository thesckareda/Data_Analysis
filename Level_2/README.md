# data_analysis
This is level 2 Data Analyzer.

In this level it reads the data through LLM api-key and by choosing required model, we can ask some intermediate level questions related to the data, it provides the desired output like:
- number of rows and columns
- names of rows and columns
- sort by rank, employment rate
- average rank, impact score, employment rate
- show highest ranked institute, institution type, by region, and many more.

In this project we have upgraded to more than basic grade by using LLM which understands in natural language and answers efficiently.
